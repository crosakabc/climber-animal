## 前提条件

すべてのファイルとフォルダをゲームのルートディレクトリに配置します：

```
D:\steam\steamapps\common\Climber Animals Together\
```

## 1. MelonLoader のインストール

> **注意**: 現在のフォルダには MelonLoader が既に含まれています。通常は再インストール不要です。  
> ローカルファイルの場所を参照し、MelonLoader プログラムフォルダをゲームのルートディレクトリに配置してください。

### 互換性に関する注意

現在の Climber Animals Together - AsianMode バージョン（最新：2026/3/30）のアップデート後も、この MOD ツールを使用できます。

**エラー発生時のトラブルシューティング**：

1. `.cs` ファイルを設定してコンパイルを試みる（**元の release ファイルを必ずバックアップしてください**）
2. ゲームを再起動
3. それでも解決しない場合は、MelonLoader を削除して最新バージョンを再インストール
4. ゲームを再起動 — 通常これで解決します

### 新規インストール手順

#### 1.1 MelonLoader 公式インストーラーのダウンロード

- ダウンロードリンク：https://github.com/LavaGang/MelonLoader/releases/download/v0.7.2/MelonLoader.Installer.exe

#### 1.2 インストーラーの実行

`MelonLoader.Installer.exe` を右クリック → **管理者として実行**

#### 1.3 ゲーム実行ファイルの選択

**Select** をクリックし、ゲーム実行ファイルを選択 — 自動的に `Climber Animals Together` が認識されます

#### 1.4 Install をクリック

**Install** をクリックすると、インストーラーが自動的に以下を実行します：

- 正しいバージョンの Cpp2IL をダウンロード（依存 DLL、完全な Plugins フォルダを含む）
- `Mods`、`Plugins`、`UserData` などのフォルダを自動生成

### 再インストール時の注意

インストールプラグイン起動時にいくつかのファイルが生成されます。再インストール時にエラーが発生する場合は、以下の手順で対応してください：

1. AI を使ってエラーを解決
2. 解決しない場合は、プラグインが生成したすべてのファイルを削除
3. exe インストーラーを再実行して必要な依存ファイルを生成

## 2. Mods ファイルについて

### 2.1 ファイルの配置

1. `AnimalTransferMod.dll` をゲームルートディレクトリの **Mods** フォルダに配置してください
2. 座標データ保存用の `AnimalTransferMod_Save.txt` が Mods フォルダに存在することを確認してください（提供されたフォルダに既に含まれています）

**パスの例**：

```
D:\steam\steamapps\common\Climber Animals Together\Mods\
```

### 2.2 その他の Mod ファイルの管理

- 他の Mod ファイルは `Mods` フォルダに残しておくことができます
- または `AnimalTransferMod.dll` と一緒に、自作の `release` フォルダにバックアップとして保存することもできます

## 3. 実装された機能

### テレポート GUI メニュー

| ショートカット | 機能 |
|----------------|------|
| **F4** | テレポート GUI メニューを開く / 閉じる |

### 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | **全プレイヤーのリアルタイム座標表示** | 全オンラインプレイヤーの位置をリアルタイム表示 |
| 2 | **自分をターゲットにテレポート** | 自分のキャラクターをターゲットプレイヤーの位置にテレポート |
| 3 | **ターゲットを自分にテレポート** | ターゲットプレイヤーを自分の位置にテレポート |
| 4 | **カスタム座標テレポート** | X/Y/Z 座標を入力して正確にテレポート |
| 5 | **位置セーブ / ロールバック** | `Ctrl+S` でセーブ / `Ctrl+Z` でロールバック |

### ⭐ コア機能：位置セーブ / ロールバック

> **安定した位置ロールバック機能**が実装され、理論上無限にリトライ可能で、**実績解除にも影響しません**。

- **セーブ**: `Ctrl+S` を押して現在のキャラクター位置を `AnimalTransferMod_Save.txt` に保存
- **ロールバック**: `Ctrl+Z` を押して最後に保存した座標にキャラクターをテレポート

## 4. 既知のバグ

| # | 問題 | 回避策 |
|---|------|--------|
| 1 | ネットワークの影響でテレポート後に**すぐに元の位置に戻ることがある** | テレポート機能を複数回試してください |
| 2 | ロールバックテレポート後にキャラクターが**動けなくなる**ことがある | ジャンプしてみてください — 通常これで回復します |
| 3 | GUI を開きっぱなしにすると**カクつく** | GUI を開きっぱなしにしないでください（座標の読み書きはフレーム単位で実行されます） |
